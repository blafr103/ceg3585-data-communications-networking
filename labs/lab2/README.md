# Lab 2 — Fourier Series and Network (Séries de Fourier et réseau)

## Objective

Two parts. First, find and program the Fourier series of each function in
Table 1 and plot the resulting curve on the local machine. Second, implement a
client-server system that sends the function data representing that curve over
the network, so the curve can be displayed on a remote machine.

## Requirements

- Use A = 10 and T = 2. For each function in Table 1, find the series by
  computing its coefficients.
- Write a program that evaluates each Fourier series up to the 100th harmonic.
- Verify the coefficients (a0, an, bn) against Desmos
  (https://www.desmos.com/calculator).
- May be implemented in C, C++, Java, or Python. C users may use CodeBlocks,
  which is available for download in the Lab2 section.

## Table 1 - functions

1. Square wave
2. Rectangular pulse train
3. Sawtooth wave
4. Triangular wave
5. Half-wave rectified sine
6. Full-wave rectified sine

d. Screenshots demonstrating the application.
e. Discussion.
f. Conclusion.
